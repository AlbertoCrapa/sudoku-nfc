# Sudoku NFC

A mobile app that lets you write Sudoku puzzles to NFC tags and play them by scanning.

## Features

- **Write** – Generate puzzles, enter them manually, or paste a string, then write to an NFC tag
- **Play** – Scan an NFC tag and start solving

## Getting Started

```bash
npm install
npx expo start
```

## Requirements

- Device with NFC capability
- NFC tags (NTAG213 or similar)
